Dans ce dossier, j'ai mis les images intermédiaires pour la création des bases de données pour le TP.

Les images originales sont dans les dossiers DB1 et DB2. Ces images proviennent des bases de données …

Ces images ont ensuite été traitées grâce au package python fingerprint_enhancer, et stockées dans les dossiers DB1_enchanced et DB2_enhanced.

Elles ont été par la suite redimensionnées pour être carrées, et stockées dans les dossiers DB1_square et DB2_square.

Parmi ces 2 dossiers, 20 images ont été sélectionnées pour créer la base de données FBI du TP (DB_FBI).

De ces 20 images, 5 ont été sélectionnées pour créer les images suspects (images_suspects_enchanced).

Enfin, ces images suspects ont été modifiées (à la main et artificiellement) pour créer nos 5 images bruitées du TP (images_suspects).

Les 2 notebooks du dossier sont pour le bruitage des images suspects, et des tests de correction de ces images bruitées.
